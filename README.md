## 🍷 Wine Cellar

> 와인 컬렉션 관리 앱 — 매장 중복 구매 방지 + 라벨 자동 인식 + 시세 추적

**기능**
- 📷 라벨 사진 → 자동 인식 (Gemini Vision, 한·영 도멘명 모두 OK)
- 🔍 한국어/영어 이름 검색으로 자동 등록
- 🏪 매장 모드: 라벨 찍으면 즉시 "보유 N병" 표시
- 🍷 시음 적기 자동 산정 (지역·등급별 drinking window)
- ⭐ Great Vintage 자동 표시 (보르도 / 부르고뉴)
- 🏰 도멘별 그룹핑 / AOC 자동 분류
- 📊 통계 (지역/색상/등급별 분포)
- 💰 자산 가치 합계
- 🔔 위시리스트
- ⭐ 시음 노트 (점수/페어링/동행자)
- 📤 CSV 내보내기
- 👥 다중 디바이스 sync (가족 공유 가능)

**다운로드**
- 📱 [Android APK v0.2.3 (86MB)](https://github.com/hykkh/h-programs/releases/download/v0.2.3/wine-cellar-v0.2.3.apk)
- 📋 [Release 노트](https://github.com/hykkh/h-programs/releases/tag/v0.2.3)
- 🌐 PWA: 준비 중

**설치 방법 (Android)**
1. APK 다운로드
2. 폰에서 탭 → "출처 알 수 없는 앱 허용" 설정 안내 따라 허용
3. 설치
4. 닉네임/비번 만들거나 시연 계정으로 로그인

**시연 계정**
- 닉네임: `demo`
- 비번: `DemoPassword2026!`
- 와인 9병 미리 등록되어 있음 (Margaux/Latour/Mouton x2/Haut-Brion + 부르고뉴 4)
