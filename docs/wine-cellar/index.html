📲 설치 방법

  1. 위 버튼으로 APK 다운로드
  2. 다운로드 알림 탭 → "설치"
  3. "출처 알 수 없는 앱 허용" 묻으면 → 허용
  4. 설치 → 앱 실행
  5. 닉네임/비번 가입 또는 시연 계정 로그인

✨ 주요 기능

  • 📷 라벨 자동 인식 (Gemini Vision)
  • 🖼️ 갤러리 단일/다중 일괄 등록 (최대 20장)
  • 🔍 한국어/영어 이름 검색 + 자동 채우기
  • 🏪 매장 모드 — 보유 여부 즉시 확인
  • 🍷 시음 적기 자동 산정 (지역·등급별)
  • ⭐ Great Vintage 자동 표시
  • 📊 통계 (지역/색상/등급별 분포)
  • 💰 자산 가치 합계
  • 🔔 위시리스트
  • ⭐ 시음 노트 (점수/페어링/동행자)
  • 📤 CSV 내보내기
  • 📸 사진 클라우드 보관
  • 👥 다중 디바이스 sync (Supabase 클라우드)

📱 사용 시나리오

  • 매장에서: 카메라로 라벨 찍기 → "보유 N병" 즉시 표시
  • 셀러에서: 갤러리 다중 선택 → 일괄 등록
  • 시음 후: 점수/페어링 음식 기록
  • 여러 폰: 같은 계정 로그인 시 데이터 동기화
